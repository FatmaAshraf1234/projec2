# Backend Project - User Authentication

## وصف المشروع
هذا المشروع عبارة عن Backend API لإدارة المستخدمين، يحتوي على:

- تسجيل مستخدمين (Register)
- التحقق من البريد الإلكتروني لمنع التكرار
- حماية كلمات المرور (Password hashing) 
- MongoDB كقاعدة بيانات
- Express.js كـ Framework

## المتطلبات
قبل تشغيل المشروع، تأكدي من وجود:

- Node.js >= 18
- MongoDB شغال على جهازك
- npm أو yarn

## خطوات تشغيل المشروع
- npm install express mongoose dotenv jsonwebtokec bcrypt
- node src/app.js
